# Moon Releases

This repository contains **public releases only** for the Moon game development AI agent.

## About Moon

Moon is a powerful game development AI agent specializing in building complete games. With expert knowledge of Unity, Unreal Engine, and Godot, Moon can handle all game development tasks from concept to deployment.

## Installation

### macOS

**Homebrew:**
```bash
brew install MoonForgeAI/tap/moon
```

**npm:**
```bash
npm install -g @moonforge/moon
```

### Windows

**Winget:**
```powershell
winget install moon
```

**Scoop:**
```powershell
scoop bucket add moonforge https://github.com/MoonForgeAI/scoop-bucket
scoop install moon
```

**npm:**
```powershell
npm install -g @moonforge/moon
```

### Linux

**Homebrew:**
```bash
brew install MoonForgeAI/tap/moon
```

**npm:**
```bash
npm install -g @moonforge/moon
```

**Arch Linux (AUR):**
```bash
yay -S moon-bin
```
